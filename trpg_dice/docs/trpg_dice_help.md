# TRPG 骰子系统使用手册

## 概述

这是一个功能完善的TRPG骰子系统插件，兼容 NekroAgent v1.0+ 插件系统，支持多种TRPG规则和复杂骰子表达式，为跑团提供全面的掷骰支持。

## 基础掷骰指令

### 基本掷骰
- `r <表达式>` - 基础掷骰
  - `r d20` - 掷一个20面骰子
  - `r 3d6+2` - 掷3个6面骰子加2
  - `r 2d10+5` - 掷2个10面骰子加5
  - `r d4+d6+d8` - 掷多种骰子组合
  - `r 3d6*5` - 乘法表达式
  - `r (2d6+6)*5` - 括号复合表达式
  - `r 4d6k3` - 保留最高的3个骰子

### 隐藏掷骰
- `rh <表达式>` - 隐藏掷骰（只显示结果，不显示过程）
  - `rh d20+5` - 暗骰，只显示最终结果

## 优势/劣势系统（D&D 5E）

### 优势掷骰
- `adv [表达式]` - 优势掷骰（掷两次取高）
  - `adv` - 优势d20
  - `adv d20+5` - 优势d20+5

### 劣势掷骰
- `dis [表达式]` - 劣势掷骰（掷两次取低）
  - `dis` - 劣势d20
  - `dis d20+3` - 劣势d20+3

## 技能检定系统

### 基础技能检定
- `ra <技能>` - 技能检定（使用角色卡中的技能值）
  - `ra 侦察` - 使用角色卡中的侦察技能
  - `ra 困难侦察` - 困难检定（技能值减半）
  - `ra 极难侦察` - 极难检定（技能值除5）

### 奖励/惩罚骰（COC7）
- `ra b<数量> <技能>` - 奖励骰
  - `ra b2 手枪` - 2个奖励骰
- `ra p<数量> <技能>` - 惩罚骰
  - `ra p1 侦察` - 1个惩罚骰

### 隐藏技能检定
- `rah <技能>` - 隐藏技能检定（只显示成功/失败）
  - `rah 侦察` - 暗中进行侦察检定
  - `rah 困难侦察` - 困难隐藏检定

### 对抗检定
- `rav <技能1> vs <技能2>` - 对抗检定
  - `rav 心理学 vs 欺骗` - 心理学对抗欺骗
  - `rav 心理学 80 vs 欺骗 60` - 指定数值的对抗检定

## 角色扮演系统

### 角色动作
- `me <动作>` - 角色动作描述
  - `me 小心翼翼地推开房门`
  - `me 向前冲锋，挥舞着剑`

## 角色卡管理系统（st）

### 查看角色卡
- `st` - 显示当前角色卡简要信息
- `st show` - 显示角色卡详细信息

### 创建和切换角色卡
- `st new <角色名>` - 创建新角色卡
  - `st new 艾莉娅` - 创建角色卡
- `st set <角色名>` - 切换到指定角色卡
- `st list` - 列出所有角色卡
- `st clr` - 清空角色卡

### 设置角色属性
- `st <属性名> <数值>` - 设置属性或技能
  - `st 力量 65` - 设置力量属性为65
  - `st 侦察 75` - 设置侦察技能为75
  - `st 侦察 +5` - 增加侦察技能5点
- `st del <属性>` - 删除属性或技能

### 模板和初始化
- `st temp coc7` - 切换到COC7模板
- `st temp dnd5e` - 切换到DND5E模板
- `st init` - 自动生成角色属性

## TRPG规则系统

### 克苏鲁的呼唤（CoC7）

#### 直接CoC检定
- `coc <技能值> [困难/极难]` - 直接CoC技能检定（无需角色卡）
  - `coc 60` - 普通检定（技能值60）
  - `coc 75 困难` - 困难检定
  - `coc 80 极难` - 极难检定

#### 理智检定
- `sc <成功损失>/<失败损失>` - 理智检定
  - `sc 1/1d4` - 成功损失1点，失败损失1d4点
  - `sc 0/1d6` - 成功无损失，失败损失1d6点

#### 技能成长
- `en <技能>` - 技能成长检定
  - `en 侦察` - 侦察技能成长

#### 疯狂症状
- `ti` - 临时疯狂症状
- `li` - 总结疯狂症状

#### 疯狂症状统一入口
- `madness [类型]` 或 `mad [类型]` - 随机疯狂症状
  - `madness` - 随机疯狂症状（临时或总结）
  - `mad 临时` - 临时疯狂症状
  - `mad 总结` - 总结疯狂症状
  - `mad 不定` - 不定疯狂症状

#### CoC角色生成
- `cocchar` 或 `coc` - 自动生成CoC角色
  - 自动生成3d6*5的属性

### D&D 5E 系统

#### 能力检定
- `check <属性/技能> [DC] [熟练]` - DND能力检定
  - `check 察觉` - 察觉检定
  - `check 力量 15` - 力量检定 vs DC15
  - `check 敏捷 15 熟练` - 敏捷检定（带熟练加值）

#### 豁免检定
- `save <属性> [DC] [熟练]` - DND豁免检定
  - `save 体质` - 体质豁免
  - `save 智力 15 熟练` - 智力豁免（带熟练加值）

#### D&D角色生成
- `dndchar` 或 `dnd` - 自动生成D&D 5E角色
  - 4d6k3生成属性

### 黑暗世界（WoD）

#### 骰池检定
- `wod <骰池大小> [困难度]` 或 `w <骰池大小> [困难度]`
  - `wod 5` - 5个d10，困难度6
  - `wod 6 8` - 6个d10，困难度8

## 生命值和状态管理

### 生命值管理
- `hp` - 查看当前生命值
- `hp +<数值>` - 恢复生命值
  - `hp +5` - 恢复5点生命值
- `hp -<数值>` - 损失生命值
  - `hp -8` - 损失8点生命值
- `hp <当前值>/<最大值>` - 设置生命值
  - `hp 25/30` - 设置生命值为25/30

## 战斗和先攻系统

### 先攻检定
- `init [+修正] [优势/劣势]` - 先攻检定
  - `init` - 基础先攻
  - `init +2` - 先攻+2
  - `init 优势` - 优势先攻
  - `init +3 劣势` - 先攻+3劣势

### 先攻追踪
- `ri` - 当前角色加入先攻
- `ri +<角色名>` - 添加角色到先攻列表
- `ri list` - 查看先攻顺序
- `ri next` - 下一回合
- `ri clear` - 清空先攻列表

## 高级掷骰功能

### 多次掷骰
- `repeat <次数> <表达式>` 或 `rn <次数> <表达式>`
  - `repeat 3 d20+5` - 掷3次d20+5
  - `rn 5 3d6` - 掷5次3d6

### 爆炸骰（开放性骰子）
- `explode <表达式>` 或 `ex <表达式>` - 爆炸骰
  - `ex d10` - d10爆炸骰（掷出10继续掷）
  - `explode 2d6` - 2个d6爆炸骰

### Fate 骰子
- `fate [修正]` 或 `f [修正]`
  - `fate` - 4个Fate骰子（默认）
  - `fate +2` - 4个Fate骰子+2

## 战报系统

### 自动模式（推荐）
系统会在第一次游戏操作（投骰、检定、角色动作）时自动开始记录，无需手动命令。

### 手动模式
- `session start [名称]` - 手动开始记录跑团会话
  - `session start 深海古城探险`
- `session event <描述>` - 记录关键事件
  - `session event 发现神秘地下入口`
- `session end` - 结束并生成战报

战报包含：玩家评分（5星制）、游戏统计、关键事件回顾、精彩时刻、Markdown格式文档。

## 文档系统

### 上传文档
- `doc_text <类型> <文档名> <内容>` - 上传文本文档
  - `doc_text module 深海古城 [模组内容...]`
  - 类型: module(模组) / rule(规则) / story(故事) / background(背景)

### 搜索和问答
- `doc search <关键词>` - 搜索文档内容
- `doc list` - 列出所有文档
- `ask <问题>` - 基于文档的智能问答

## 辅助功能

### 随机姓名生成
- `name [参数]` - 随机姓名生成
  - `name` - 随机中文姓名
  - `name en` - 英文姓名
  - `name 男` - 男性姓名
  - `name 女 3` - 生成3个女性姓名
  - `name en male 2` - 生成2个英文男性姓名

### 今日人品
- `jrrp` - 今日人品检定
  - 基于用户ID和日期生成固定的日运值
  - 包含人品等级判定

### 抽卡功能
- `draw [数量]` - 抽取标准扑克牌
  - `draw` - 抽1张牌
  - `draw 5` - 抽5张牌

### 机器人状态
- `bot` - 查看机器人状态和版本信息

### 帮助
- `help` - 查看TRPG系统帮助

## 骰子表达式语法

### 基础语法
- `d<面数>` - 单个骰子（如 d20, d6）
- `<数量>d<面数>` - 多个骰子（如 3d6, 2d10）
- `+<数值>` / `-<数值>` - 加减修正值（如 +5, -2）

### 复合表达式
- `3d6+2d4+5` - 多种骰子组合
- `d20+d6+3` - 不同骰子和修正值
- `(2d6+6)*5` - 括号乘法

### 特殊规则
- **优势**：掷两次d20取最高
- **劣势**：掷两次d20取最低
- **爆炸骰**：掷出最大值时继续掷骰（最多10次）
- **Fate骰子**：结果为-1、0、+1

## 成功程度判定

### CoC系统成功程度
- **大成功**：掷出1
- **极难成功**：≤技能值/5
- **困难成功**：≤技能值/2
- **成功**：≤技能值
- **失败**：>技能值
- **大失败**：掷出100，或技能<50时掷出96-100

### 对抗检定规则
1. 双方都成功：比较成功程度
2. 成功程度相同：骰点低者胜
3. 仅一方成功：成功方胜利
4. 双方都失败：骰点低者胜

## 大成功/大失败

系统自动检测并提示：
- ✨ 大成功：掷出骰子最大值（单骰检定时）
- 💥 大失败：掷出1（单骰检定时）

## 角色卡系统详解

### CoC角色卡
- **主属性**：力量、体质、体型、敏捷、外貌、智力、意志、教育、幸运
- **派生属性**：理智、魔法、生命值、体力
- **技能系统**：包含所有CoC标准技能（闪避=DEX/2，母语=EDU）

### D&D 5E角色卡
- **六大属性**：力量、敏捷、体质、智力、感知、魅力
- **派生属性**：生命值、护甲等级、先攻修正、熟练加值
- **技能系统**：基于属性修正值 + 熟练加值

## 配置选项

插件支持以下配置（在NekroAgent WebUI中调整）：
- `MAX_DICE_COUNT`: 单次最大骰子数量（默认100）
- `MAX_DICE_SIDES`: 骰子最大面数（默认1000）
- `DEFAULT_DICE_TYPE`: 默认骰子类型（默认d20）
- `ENABLE_CRITICAL_EFFECTS`: 启用大成功/大失败（默认开启）
- `ENABLE_VECTOR_DB`: 启用文档向量化存储（默认开启）
- `CHUNK_SIZE`: 文档分块大小（默认1000）
- `CHUNK_OVERLAP`: 分块重叠大小（默认200）
- `MAX_SEARCH_RESULTS`: 最大搜索结果数（默认5）

## 注意事项

1. 角色数据会自动保存到NekroAgent存储系统
2. 大成功/大失败的判定只在单骰检定时生效
3. 爆炸骰有防无限循环机制（最多10次）
4. 支持中文属性名和技能名，中英文别名自动识别
5. 今日人品值基于用户ID和日期生成，每日固定
6. 战报系统会在第一次游戏操作时自动启动
7. 文档内容按聊天会话隔离，保障安全性

这个TRPG骰子系统提供了完整的跑团支持，能够满足COC7、DND5E、WoD等主流TRPG系统的需求。
